Conventional Commits

This project uses Conventional Commits. For an interactive commit message form, use git cz as a drop in replacement for git commit. If git commit is preferred, a git-hook to lint the message can be used. Both tools require NodeJS and node_modules/.bin will need to be in the shell’s path.

npm install --production
husky install
npx husky add .husky/commit-msg "npx --no-install commitlint --edit $1"
